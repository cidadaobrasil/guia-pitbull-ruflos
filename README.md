# Guia Pit Bull do Ruflos

Manual em português para a criação responsável do Ruflos por Joás e Laiza.

O site é um HTML estático, pronto para GitHub Pages, com:

- 8 fases de criação e manejo;
- capítulos adicionais sobre comportamento, saúde, emergências e equipamentos;
- busca, favoritos, checklist diário e modo impressão/PDF;
- referências de AVSAB, AVMA, AAHA, Merck Veterinary Manual e autores de comportamento.
